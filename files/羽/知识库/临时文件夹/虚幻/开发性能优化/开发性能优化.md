1. 项目-目标硬件 设置成移动设备和可缩放
2. 项目设置-渲染
	1. 动态全局光照：无 或 屏幕空间
	2. 反射方法：无 或 屏幕空间
	3. 软件光线追踪模式：全局追踪
	4. 阴影贴图方法：阴影贴图
	5. 抗锯齿方法：TAA或无
3. 预览模式选移动/mobile (会影响地形创建)
4. 关掉实时
5. 如果添加了灯光，没有烘焙的话，编辑器用的是间接光照缓存，预烘焙，资源消耗很大
6. 引擎-一般设置 使用固定帧率
